Palindrome Checker

Explanation for console code

The code checks if a number is a palindrome by following these steps:

  • Convert the Number to a String: Convert the number to a string to work with its digits individually.
  • Reverse the String: Reverse the string to compare it with the original.
  • Compare the Original String with the Reversed String: If they are identical, the number is a palindrome.

Here's the code:

function isPalindrome(number) {
    const str = number.toString();
    const reversedStr = str.split('').reverse().join('');
    return str === reversedStr;
}

const number = document.getElementById('number').value;

if (isPalindrome(number)) {
    console.log("The number ", number, " is Palindrome number");
} else {
    console.log("The number ", number, " is not Palindrome number");
}
